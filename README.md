# enerOS | Electrical Energy Operating System

enerOS is a modular energy software platform for visibility, intelligence, and coordination across distributed power assets.

## Current stage

The project is in early product definition and prototype setup.

## MVP

enerOS Site Intelligence MVP: monitor and analyze one facility or simulated energy site with solar PV, battery storage, generator, grid supply, and electrical loads.

## Repository structure

- website/ - product/brand website
- docs/ - product and engineering documentation
- apps/dashboard/ - future operator dashboard
- apps/api/ - backend API
- services/simulator/ - simulated telemetry generator
- data/ - sample telemetry files
