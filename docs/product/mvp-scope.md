# enerOS MVP Scope

Project: enerOS  
Full name: Electrical Energy Operating System  
MVP name: enerOS Site Intelligence MVP  

## Goal
Monitor and analyze one facility or simulated energy site with solar PV, battery storage, generator, grid supply, and electrical loads.

## Primary user
Commercial/industrial facility energy operator.

## Main value
Give operators visibility into distributed energy assets and support better energy decisions.

## Core features
- Dashboard
- Telemetry simulation
- Asset registry
- Alerts
- Historical trends
- Basic analytics
- Recommendations

## Excluded from MVP
- Fully automated dispatch
- Utility-wide DERMS
- Complex forecasting
- Billing optimization
- Hardware certification
