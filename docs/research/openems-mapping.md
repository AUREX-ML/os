# OpenEMS-to-enerOS Architecture Mapping

OpenEMS is being studied as a reference model for the enerOS architecture. Its modular design, device abstraction patterns, and protocol support provide useful reference points as enerOS is designed and built.

## Concept Mapping

| OpenEMS concept | enerOS equivalent |
| --- | --- |
| OpenEMS Edge | enerOS Edge Gateway |
| OpenEMS UI | enerOS Operator Dashboard |
| OpenEMS Backend | enerOS Cloud Coordination Layer |
| Device abstraction | enerOS Asset Model |
| Modular controllers | enerOS Control/Optimization Modules |
| Supported protocols | enerOS Integration Layer |
